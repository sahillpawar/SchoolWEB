# Quick Start Guide

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```
   Open http://localhost:5173 in your browser

3. **Build for Production**
   ```bash
   npm run build
   ```

4. **Deploy to GitHub Pages**
   ```bash
   npm run deploy
   ```

## Customization Checklist

Before deploying, customize these areas:

- [ ] **Projects** - Update project data in `src/components/Projects/Projects.jsx`
- [ ] **YouTube Videos** - Replace video IDs in `src/components/MediaGallery/MediaGallery.jsx`
- [ ] **Images** - Replace placeholder images with your own
- [ ] **Contact Info** - Update email, phone, and social links in `src/components/Contact/Contact.jsx`
- [ ] **About Content** - Customize the About section text
- [ ] **Learning Topics** - Update learning resources in `src/components/Learning/Learning.jsx`
- [ ] **Colors** - Modify CSS variables in `src/styles/global.css` if needed
